NetBeans
Qu'est-ce qu'un Apache NetBeans ?

NetBeans est un environnement de développement intégré (EDI), placé en open source par Sun en juin 2000 sous licence CDDL (Common Development and Distribution License) et GPLv2. En plus de Java, NetBeans permet la prise en charge native de divers langages tels le C, le C++, le JavaScript, le XML, le Groovy, le PHP et le HTML, ou d'autres (dont Python et Ruby) par l'ajout de greffons. Il offre toutes les facilités d'un IDE moderne (éditeur avec coloration syntaxique, projets multi-langage, refactoring, éditeur graphique d'interfaces et de pages Web).

Télécharger Appache NetBeans

IntelliJ IDEA
Qu'est-ce qu'IntelliJ IDEA ?

IntelliJ IDEA également appelé « IntelliJ », « IDEA » ou « IDJ » est un environnement de développement intégré (en anglais Integrated Development Environment - IDE) destiné au développement de logiciels informatiques reposant sur la technologie Java. Il est développé par JetBrains (anciennement « IntelliJ ») et disponible en deux versions, l'une communautaire, open source, sous licence Apache 2 et l'autre propriétaire, protégée par une licence commerciale. Tous deux supportent les langages de programmation Java, Kotlin, Groovy et Scala.

Télécharger Intellij IDEA

Visual Studio
Qu'est-ce qu'un Visual studio ?

Microsoft Visual Studio est une suite de logiciels de développement pour Windows et mac OS conçue par Microsoft. VS Code est la capacité de déboguer les applications directement sur l'éditeur sans même avoir recours aux navigateurs, grâce à un système de points d'arrêt et une console de débogage intégrée qui permet de résoudre les problèmes directement dans l'éditeur. c'est une Plateforme de développement d'applications et de jeux qui permet aux concepteurs de les développer, modifier et tester grâce à divers outils et techniques. L'une des fonctionnalités les plus importantes de Visual Studio est IntelliSense : un assistant de complétion de code qui vous accompagne en listant chaque ligne de code que vous produisez

Télécharger Visual Studio

Cloud9 IDE

Qu'est-ce qu'un Cloud9 ?

Cloud9 est une plateforme de développement en ligne qui rend le développement d'applications plus pratique que jamais Cloud9 combine un puissant éditeur de code en ligne avec un espace de travail Ubuntu complet dans le cloud. Il prend en charge plus de 40 langues, avec une prise en charge de classe A pour PHP, Ruby, Python, JavaScript, Go, etc. Les espaces de travail Cloud9 sont alimentés par des conteneurs Docker Ubuntu qui vous donnent une liberté totale sur votre environnement, y compris les droits sudo.
